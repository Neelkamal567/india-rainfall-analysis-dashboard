# 🌧 India Rainfall Analysis Dashboard

## 📊 Project Overview
This project analyzes rainfall patterns across India from 1901 to 2017 using Microsoft Excel.  
An interactive dashboard was created to explore trends, regional variations, and seasonal distribution.

---

## 🔍 Key Insights
- 🌧 Monsoon contributes ~75% of total rainfall  
- 🌍 Wettest Region: Arunachal Pradesh  
- ☀️ Driest Region: West Rajasthan  
- 📈 Significant year-to-year rainfall variability  

---

## 🛠 Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers)  
- Data Visualization Techniques  
- Overleaf (IEEE Report)

---

##  Dataset
- Source: Government of India Open Data Portal  
- IMD Subdivision Rainfall Data (1901–2017)

## 📌 Features
- KPI Cards (Avg Rainfall, Regions, Monsoon Insights)  
- Trend Analysis (Annual & Monsoon)  
- Regional Comparison (Wettest & Driest Regions)  
- Monthly & Seasonal Distribution  
- Interactive Filters (Slicers)

---

## 🚀 How to Use
1. Download the Excel file  
2. Open dashboard sheet  
3. Use slicers to filter by year and region
